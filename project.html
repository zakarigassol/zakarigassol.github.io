HERE ARE SOME OF MY PROJECTS

Online Delivery Application

Collaborated with a four-member team to design and develop a delivery app using Flutter and Firebase.
The app allows users to register, log in, and request delivery services, while providing drivers with delivery assignments.
The project included user registration, real-time data synchronization, and a secure authentication system.
An admin section was implemented to manage the database and track deliveries.
Leveraged UML diagrams for system design, employed Dart as the primary programming language, and ensured cross-platform functionality for both Android and iOS devices.
Delivered a userfriendly and secure delivery platform.

Horse Riding Club Website

Developed a membership website for a horse-riding club using PHP to streamline the registration and management of members.
The platform allows visitors to browse membership plans, users to register and manage memberships through personalized dashboards, and admins to efficiently manage user and membership data via an admin panel.
Implemented features for user registration, real-time data handling, and admin control over membership plans.
Focused on enhancing user experience, improving communication, and reducing administrative tasks. Integrated a MySQL database for efficient data management.

Cambridge Hospital Website

As part of a team of five students at Aptech, I contributed to the development of a fully functional, responsive hospital website.
The project showcased proficiency in HTML5, CSS3, JavaScript, and Bootstrap.
My role included writing some HTML5 code for the front end.
Key features of the website include doctor scheduling, feedback submission, appointment booking, and geolocation integration for finding hospital locations.
The website is optimized for fast loading and scalability,utilizing tools such as Google Fonts, Animate.css, and online image optimization.
